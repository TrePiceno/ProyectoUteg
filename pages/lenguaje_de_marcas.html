El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

HTML es la columna vertebral de cualquier página web. A pesar de su simplicidad, ofrece una gran flexibilidad y control sobre la estructura y contenido de la web. Entender y utilizar eficazmente HTML es esencial para cualquier desarrollador frontend, ya que es la base sobre la cual se construyen y estilizan las aplicaciones web.
